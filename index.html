Project on Console

The Visual Studio IDE (integrated development environment) is a software program for developers to write and edit their code. Its user interface is used for software development to edit, debug and build code.

What is console in Visual Studio?

Console application (app) is a program developed in Visual Studio which accepts input parameter, calls the required service, runs business logic and sends output to the console, this console is known as the command prompt.

How to open Console?

Follow the steps to open console.

1.Right click on the page

2.Click on inspect

3.Click on Console
